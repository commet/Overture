# Overture

**Think before you orchestrate.**

AI에게 일을 시키는 건 쉬워졌습니다.
어려운 건 **무슨 일을 시킬지 정하는 겁니다.**

Overture는 AI에게 일을 시키기 전에 거쳐야 할 **판단의 구조**를 만들어주는 도구입니다.

---

## 왜 필요한가

ChatGPT에 "경쟁사 분석해줘"라고 바로 시키면, 결과는 그럴듯하지만 의사결정에 쓸 수 없는 경우가 많습니다. **질문 자체가 틀렸기 때문입니다.**

LangGraph, CrewAI 같은 도구는 AI 에이전트를 **어떻게 실행할지**를 다룹니다.
Overture는 **무엇을 실행해야 하는지**를 다룹니다.

```
Without Overture:   과제 → 바로 AI → 그럴듯하지만 쓸 수 없는 결과
With Overture:      과제 → 판단의 구조 → AI 실행 → 의사결정에 쓸 수 있는 결과
```

---

## 4단계 사고 프로세스

| 단계 | 이름 | 하는 일 |
|------|------|---------|
| 1 | **악보 해석** · 문제 재정의 | 과제의 숨겨진 가설, 전제, 진짜 질문을 찾습니다 |
| 2 | **편곡** · 실행 설계 | AI/사람 역할을 나누고 워크플로우를 설계합니다 |
| 3 | **리허설** · 사전 검증 | 이해관계자 시점에서 결과물을 미리 검증합니다 |
| 4 | **합주 연습** · 피드백 반영 | 검증 결과를 반영하여 수렴할 때까지 반복합니다 |

## 실제 사용 예시

**상황**: 대표가 "경쟁사가 AI 챗봇 출시했으니 우리도 빨리 만들어"라고 지시했다.

### 1. 악보 해석
카드 선택 3번 + 한 문장 입력 → AI가 분석:
> "챗봇을 만들어야 하는가?"가 아니라
> **"고객 이탈의 진짜 원인이 뭔가?"**가 먼저 답해야 할 질문입니다.

숨겨진 가설, 검증 안 된 전제, 대안적 관점도 함께 제시됩니다.

### 2. 편곡
AI가 7단계 워크플로우를 자동 설계합니다. AI 트랙과 사람 트랙이 병렬로 시각화됩니다:

- 🤖 AI: 시장 데이터 수집, 고객 리뷰 분석
- 🧠 사람: 이탈 원인 해석, 최종 방향 판단
- ⚑ 체크포인트: 3단계에서 반드시 사람이 검증

각 단계의 기대 산출물, 판단 포인트, 핵심 가정, 크리티컬 패스가 명시됩니다.

### 3. 리허설
페르소나를 등록하고(프리셋 6종 또는 직접 생성) 결과물에 대한 피드백을 시뮬레이션합니다:

> **김 CFO**: "ROI 추정 없이 예산 요청 불가."
> **프리모템**: "이 계획이 실패한다면, 고객 조사 없이 솔루션을 결정했기 때문."
> **승인 조건**: "3개월 파일럿 결과와 경쟁사 대비 벤치마크를 보여주면 승인."

정확도 평가를 남기면 다음 시뮬레이션이 더 정확해집니다.

### 4. 합주 연습
리허설에서 나온 우려사항 중 해결할 것을 선택 → 제약조건으로 변환 → 1단계부터 다시 분석.
수렴률이 추적됩니다: `0% → 45% → 78% → 92%`

### 산출물
같은 사고 과정에서 목적에 맞는 4가지 형식으로 내보냅니다:

- **프로젝트 브리프** — 사람에게 공유할 의사결정 기록
- **프롬프트 체인** — Claude/ChatGPT에 순서대로 입력할 프롬프트 세트
- **Agent Spec** — LangGraph/CrewAI 구현의 출발점이 되는 설계서
- **실행 체크리스트** — 단계별 검증용 체크리스트

## 기존 도구와 다른 점

| | ChatGPT/Claude | LangGraph/CrewAI | **Overture** |
|---|---|---|---|
| 핵심 질문 | 바로 실행 | 어떻게 실행? | **무엇을 실행?** |
| 질문이 틀리면? | 그럴듯한 쓸모없는 답 | 더 빠르게 잘못된 결과 | **질문부터 교정** |
| 여러 AI 답변이 다르면? | "종합해줘" → 표면 합의 | 에이전트끼리 투표 | **쟁점별 내 판단** |
| 사용할수록? | 매번 처음부터 | 코드 수정 필요 | **자동으로 패턴 학습** |
| 이해관계자 검증? | 없음 | 없음 | **페르소나 시뮬레이션** |

---

## 사용할수록 똑똑해집니다

- **판단 기록**: 질문 선택, 역할 변경, 쟁점 판단이 자동으로 기록됩니다
- **패턴 학습**: 축적된 판단 패턴이 다음 AI 분석에 자동 반영됩니다
- **페르소나 정확도**: 실제 반응을 기록하면 시뮬레이션 정확도가 높아집니다
- **유사 분석 추천**: 과거 프로젝트와 유사한 과제를 발견하면 참고 자료로 제시합니다

배포 URL은 준비 중입니다. 아래 로컬 실행으로 먼저 사용할 수 있습니다.

### 로컬 실행

```bash
git clone https://github.com/commet/Overture.git
cd Overture
npm install
npm run dev
```

`http://localhost:3000`에서 실행됩니다.

### API 키 설정 (선택)

기본적으로 서버 프록시를 통해 작동합니다. 사용량 제한 없이 사용하려면:

1. 설정 페이지에서 "직접 API 키" 모드 선택
2. Anthropic API Key 입력
3. 키는 브라우저 localStorage에만 저장되며 서버로 전송되지 않습니다

## 기술 스택

- **Next.js 16** (App Router) + **TypeScript**
- **Tailwind CSS v4** + **Pretendard Variable** (한국어 폰트)
- **Zustand** (상태관리 + localStorage 자동 저장)
- **Anthropic Claude API** (서버 프록시 또는 직접 호출)
- **Lucide React** (아이콘)

